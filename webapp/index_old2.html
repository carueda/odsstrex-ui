note: all of this is preliminary.
  • To add a new goal: on an empty region, double click (a dialog will appear) or click and drag to indicate the range for the goal (and a dialog will appear to enter more information).
  • To remove a goal: double click the event and select Delete in the dialog that appears.
some devel notes:

Functionality is still preliminary and currently focused on the interface composition and user interaction capabilities. Although data is already coming from a back-end servlet, it is still fake data. Not yet connection with T-REX endpoint in the back-end.
  • As can be noted the component is currently isolated from the ODSS interface to facilitate development; but the integration with ExtJs has already been proved so no issues are expected when integrating later on.
  • The shaded area in the background shows a "period of interest" -- only events overlapping with that area are reported by the back-end. This period of interest could be for example a particular experiment.
  • The dialog for event creation/editing is customizable (can add more input elements, for example), which might be useful depending on needed related information.
